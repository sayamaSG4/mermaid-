# 法人の確定申告 手順まとめ（A4簡易版）

## 1. 申告の概要

| 項目 | 内容 |
|------|------|
| 対象 | 法人（株式会社・合同会社など） |
| 申告期限 | 決算日の翌日から2か月以内（延長申請で＋1か月可） |
| 提出先 | 税務署・都道府県税事務所・市区町村役場 |
| 主な税目 | 法人税・地方法人税・法人住民税・法人事業税 |

---

## 2. 全体の流れ

```mermaid
graph TD
A[① 決算処理] --> B[② 申告書作成]
B --> C[③ 電子申告・提出]
C --> D[④ 納税]
D --> E[⑤ 保管・事後対応]

```

```mermaid

graph TD
    A[開始] --> B{シート 提出者取引明細 の存在確認}
    B -- なし --> C[エラー終了]
    B -- あり --> D[InputBox: 処理開始行の指定]
    D --> E[新規ブック作成 & 全セル書式を文字列に設定]
    E --> F[ソース行を順次ループ開始]
    F --> G{同一人物・同日判定}
    G -- 同一 かつ 4件未満 --> H[既存行の右隣スロットへ転記<br>groupCount ++]
    G -- 異なる または 5件目 --> I[新規行を作成して転記<br>destRow ++, groupCount = 1]
    H --> J[最終行まで継続]
    I --> J
    J --> K[A, D, E, F, J, M, AR, AS列に固定値を一括入力]
    K --> L[CSV形式で保存 & クローズ]
    L --> M[完了通知]

```
